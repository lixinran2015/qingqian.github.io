Hbase 搭建

  • 准备工作
    • 文件：hbase-1.1.2-bin.tar.gz
  • 正式环境
    • 设置环境变量
      • 设置文件 .bashrc
      • export HBASE_HOME=/home/hadoop/bigdata/hbase
        export PATH=$HBASE_HOME/bin:$PATH
        • HBASE_HOME 表示HBASE安装目录
      • source .bashrc
      • 验证
        • [hadoop@master ~]$ echo $HBASE_HOME
          /home/hadoop/bigdata/hbase
      • 拷贝到slave机器
        • scp .bashrc hadoop@192.168.0.167:/home/hadoop/
        • scp .bashrc hadoop@192.168.0.168:/home/hadoop/
      • 验证slave
        • 方法同master端
    • 安装目录

      • cd /home/hadoop/bigdata/
        tar -zxf hbase-1.1.2-bin.tar.gz

        mv hbase-1.1.2-bin hbase

    • hbase-env配置
      • cd /home/hadoop/bigdata/hbase/conf
      • export JAVA_HOME=/usr/java/jdk1.8.0_60
    • hbase-site配置

      • cd /home/hadoop/bigdata/hbase/conf

        <property>   <name>hbase.rootdir</name>   
        <value>hdfs://master:9000/hbase</value>   
        </property>
        <property>
        <name>hbase.zookeeper.quorum</name>
        <value>master,slave01,slave02</value>
        </property>
        <property>
        <name>hbase.cluster.distributed</name>
        <value>true</value>
        </property>
        <property>
        <name>hbase.zookeeper.property.dataDir</name>
        <value>/home/hadoop/bigdata/zk/zkdata</value>
        </property>
